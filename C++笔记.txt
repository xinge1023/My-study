无符合int类型数据永远大于等于0,最大值为4294967295,4294967296等于0
unsigned int a = -1;            a = 4294967295;
unsigned int a = 5000000000;    a = 705032704(5000000000 - 4294967296);
unsigned int a = -5000000000;   a = 3589934592(4294967296 - (5000000000 - 4294967296));
----------------------------------------------------------------------------------------
无符合char类型数据永远大于等于0,最大值为255,对应ascii码扩展表
unsigned  char a = 97;          a = 'a';
----------------------------------------------------------------------------------------
int a = 20;    int类型为十进制
int a = 024;   以0开始的int类型为八进制
int a = 0x14;  以0x开始的int类型为十六进制
----------------------------------------------------------------------------------------
int类型初始化的四种方法：
int a = 5;
int a = {5};
int a(5);
int a{5};
----------------------------------------------------------------------------------------
(非对象)引用存放的是值,只能绑定一次对象,继续绑定不影响值,能改变引用的变量值。
(对象)指针存放的是地址，可以任意绑定，不能改变引用的变量值。
可一次性定义多个int型数据变量; int a = 1024, *b = &a, &c =a;
